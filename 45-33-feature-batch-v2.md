`45-33-voice-content-v3.md`, раздел «Новая логика — единый взвешенный пул».

   - Все факты из массива лейбла, если опознан (вес 3).
   - Все факты из массива жанра (вес 2).
   - Все факты из массива эпохи (вес 2).

   `PrivacyService`): если приватный режим включён — `€•••`, иначе
   сумма. Текст рядом — из VoiceContent, ключ `SUMMARY_TOTAL_SPENT`
   с уже встроенной логикой порогов (< €500, €500–€1500, €1500+).

3. **Топ-3 жанра.** Список из трёх строк: «Джаз · 47 пластинок»,
   с маленькой полосочной диаграммой относительной длины.

4. **Топ-3 артиста.**

5. **Топ-3 лейбла.**

6. **Распределение по десятилетиям.** Бар-чарт. По оси X — десятилетия
   (1950, 1960, ..., 2020), по оси Y — количество. Использовать `Chart`
   из SwiftUI Charts (доступен с iOS 16).

7. **Сюжетные моменты:**
   - «Самая дорогая покупка года» — карточка пластинки.
   - «Самая старая» — карточка пластинки с годом.
   - «Первая в этом периоде» — карточка с датой добавления.
   - «Последняя» — карточка с датой.

8. **Достижения за период** — если есть, список с иконками.

9. **Помесячная активность.** Маленький столбчатый график 12
   столбиков (январь-декабрь или последние 12 месяцев).

10. **Кнопка «Поделиться сводкой»** внизу — генерирует PNG (см. 2.3).

**Тон:**

- Заголовки секций — короткие, в нашем стиле: «топ жанра», «сюжет
  года», «по месяцам».
- Между секциями — короткие вайбовые фразы из новых ключей в
  VoiceContent (см. 2.4).
- Не «у вас 47 пластинок жанра джаз», а «джаз прижился —
  47 пластинок».

**Точка входа:**

В `ProfileView`, после блока «Достижения» — карточка с заголовком
«Год в виниле» и подзаголовком «сводка за период». Тап → открывает
`YearlyRecapView` как push в navigation stack.

**Критерии приёмки:**
- Открытие экрана — мгновенное (расчёт быстрый, < 100мс на 200
  пластинках).
- Все цифры считаются корректно: сумма соответствует сумме `price` у
  отфильтрованных записей, топ-3 действительно три самых частых.
- Графики рендерятся корректно при пустых периодах (показывают
  placeholder, не падают).
- Сменa периода в picker'е плавно перерисовывает все секции.

## 2.3. Шаринг сводки как PNG

Создать функцию `YearlyRecapSharePoster` в `Views/YearlyRecapView.swift`
(или отдельный файл).

**Дизайн-постера для шаринга:**

- Соотношение **4:5 вертикальное** (стандарт Instagram-постов).
- Тёмный фон в стиле приложения (`AppTheme.bg`).
- Сверху — заголовок «Год в виниле · 2024» крупно, шрифт `Fraunces`
  или серифный системный.
- Большое число — количество пластинок, золотым акцентом.
- Три блока с топ-3 жанрами / артистами / лейблами в виде вертикальных
  колонок.
- Минимальная сетка-диаграмма по десятилетиям внизу.
- В нижнем углу — «45/33» как подпись (используй `SpeedMark45_33`
  из компонентов).
- Опциональный ник пользователя из `UserProfile.handle`.

Использовать `ImageRenderer` (есть в `Services/SharePNGExporter.swift` —
переиспользуй паттерн).

Кнопка «Поделиться сводкой» → генерация → `UIActivityViewController`
с готовой PNG.

**Критерии приёмки:**
- PNG-файл генерируется в высоком разрешении (минимум 1080×1350 для
  Instagram-качества).
- При отсутствии хотя бы одного раздела (например, нет лейблов
  совсем) — этот блок на постере не показывается, остальные
  смещаются.
- Шаринг открывает системный sheet — пользователь может выбрать
  Instagram / Telegram / Photos / любое приложение.

## 2.4. Тексты для «Год в виниле»

Добавить в `VoiceContent.phrases` новые ключи:

- `recapHeroEmpty` — если за период 0 пластинок: «Тихий год. С полкой
  ничего не случилось.» (и ещё 3 варианта).
- `recapHeroFew` (1-5 пластинок) — «Аккуратное начало. {N} пластинок
  за год — это уже что-то.» (и варианты).
- `recapHeroNormal` (6-30): «{N} пластинок прижились за год. Полка
  заметно потяжелела.»
- `recapHeroHeavy` (30+): «{N} пластинок за год — серьёзная охота.»
- `recapTopGenreIntro` — «жанр года» / «что чаще всего».
- `recapTopArtistIntro` — «артист года» / «к кому вернулся чаще
  всех».
- `recapTopLabelIntro` — «лейбл года».
- `recapMostExpensive` — «самая дорогая покупка».
- `recapOldestPress` — «самая старая пластинка года».
- `recapMonthlyTitle` — «по месяцам» / «когда тяжелее всего была
  полка».

По 3-5 вариантов каждый, в нашем тоне. Тон — рефлексивный, тёплый,
не подталкивающий («ты молодец, давай ещё больше»). Скорее «вот так
это выглядело», чем «вот результат».

**Файлы:** `Services/VoiceContent.swift`.
