## ИНСТРУКЦИИ ДЛЯ ИИ-КОДОГЕНЕРАТОРА · РАНДОМИЗАЦИЯ ФАКТОВ (v3)

### Проблема, которую решаем
В v2 при добавлении одной и той же пластинки пользователь мог получать факт
из одного и того же массива несколько раз подряд — например, один и тот же
жанровый комментарий про рок при пяти попытках. Это происходило потому, что
логика выбора шла «по приоритету сверху вниз», и фоллбэк всегда упирался в
один и тот же массив. Внутри массива была вариативность, но между источниками —
не было.

### Новая логика — единый взвешенный пул

Когда пользователь добавляет пластинку, ИИ должен:

**Шаг 1. Собрать пул кандидатов.** Из всех релевантных источников. Для пластинки
Yellow Submarine (The Beatles, 1969, Apple Records, рок) пул собирается так:
- Все факты из Раздела Б, где artist+title совпадают с пластинкой (если есть).
- Все факты из FACT_LABEL_APPLE (если массив существует).
- Все факты из FACT_GENRE_ROCK.
- Все факты из FACT_DECADE_1960s.
- Часть фактов из FACT_GENERIC (взять ~3 случайных, чтобы не перегружать).

**Шаг 2. Назначить веса.** Более специфичный источник — выше вес:
- Раздел Б (конкретная пластинка): вес 5
- Лейбл (если опознан): вес 3
- Жанр: вес 2
- Эпоха: вес 2
- Прессинг (если есть особенность): вес 2
- Общие (FACT_GENERIC): вес 1

**Шаг 3. Глобальная память.** Хранить ID последних 20 показанных фактов
(не только в текущей сессии — между сессиями тоже, в локальном хранилище).
При выборе **исключать из пула** все, что в этом списке. Если после
исключения пул пуст — очистить память и начать заново.

**Шаг 4. Случайный выбор с учётом весов.** Сделать единый случайный выбор
из оставшегося пула, с вероятностями пропорциональными весам.

**Шаг 5. Добавить выбранный факт в начало списка «недавно показанные».**
Удалить хвост, если список превысил 20 элементов.

### Что это даёт
- При повторных добавлениях одной пластинки каждый раз показывается **факт
  из другого источника** (сначала, скажем, факт про сам Yellow Submarine,
  потом про Apple, потом про эпоху, потом снова про конкретную пластинку,
  но уже второй факт оттуда).
- Внутри сессии и между сессиями факты не повторяются.
- Пользователь чувствует, что приложение «знает много», а не «крутит пять
  одних и тех же фраз».

### Дополнительные правила
- **Один и тот же факт никогда не показывается дважды подряд** — это
  абсолютное правило, оно сильнее всех остальных.
- **Если у пластинки в Разделе Б есть запись с несколькими фактами** — при
  выборе из этого источника тянем случайный из них, а не первый по порядку.
- **Факты показываются в 60% случаев при добавлении пластинки.** В остальных
  40% — добавление без факта, чтобы не приедалось.
- **На конвейере серийного сканирования факты выключены.** Всегда.
- **При первой пластинке в коллекции (юбилей 1)** факт обязателен — это
  «знакомство», там пропускать нельзя.

### Псевдокод
```
function pickFact(record, recentFacts):
  pool = []
  
  if exact_match_in_section_B(record):
    pool += [(fact, weight=5) for fact in section_B[record].facts]
  
  if record.label in known_labels:
    pool += [(fact, weight=3) for fact in FACT_LABEL[record.label]]
  
  if record.genre:
    pool += [(fact, weight=2) for fact in FACT_GENRE[record.genre]]
  
  if record.decade:
    pool += [(fact, weight=2) for fact in FACT_DECADE[record.decade]]
  
  if record.press_type:  # mono, japanese, colored, etc
    pool += [(fact, weight=2) for fact in FACT_PRESS[record.press_type]]
  
  pool += [(fact, weight=1) for fact in random_sample(FACT_GENERIC, 3)]
  
  # фильтруем недавно показанные
  pool = [(f, w) for (f, w) in pool if f.id not in recentFacts]
  
  if pool is empty:
    clear(recentFacts)
    return pickFact(record, [])
  
  # взвешенный случайный выбор
  chosen = weighted_random(pool)
  
  recentFacts.prepend(chosen.id)
  if len(recentFacts) > 20:
    recentFacts.pop()
  
  return chosen
```

## ОБНОВЛЁННЫЕ И НОВЫЕ ЖАНРОВЫЕ МАССИВЫ

### [FACT_GENRE_JAZZ] — расширено
```
- Джаз на виниле — это, кажется, единственный честный способ его слушать.
- Хороший выбор. Джаз и винил — почти синонимы.
- Джаз был придуман для аналогового звука. На цифре он немного теряет.
- В джазовых клубах 50-х играли с этих самых пластинок.
- Джаз и шипение винила — фоновый шум одной эпохи.
- Хороший джаз слышен пальцами — как клавиша касается струны, как палочка касается тарелки.
- Джазовый альбом редко слушают треками. Это сторонами.
- На виниле слышно дыхание трубача между нотами. На стриме — нет.
- Джазовая пластинка — это присутствие. Десятки музыкантов в одной комнате, в одно время.
- Хороший джазовый прессинг звучит так, будто инструменты стоят у тебя в гостиной.
```

### [FACT_GENRE_ROCK] — расширено
```
- Классика жанра. Эту до сих пор переиздают каждые пару лет.
- Рок на виниле — всё ещё лучший вариант. Цифра не вытягивает.
- Рок-альбомы делали для виниловых сторон по 20 минут — отсюда и структура.
- Тяжёлая гитара на виниле звучит иначе. Аналоговое искажение делает её живой.
- Виниловое сжатие звука для рока — не проблема, а особенность. Гитары становятся плотнее.
- Рок-альбомы 70-х писали с расчётом на иглу, не на наушники.
- Если бэндовый драйв есть — винил его передаёт лучше любого формата.
- Многие классические рок-пластинки звучат на виниле так, как нигде больше.
- Рок и аналоговая запись — пара, которую цифровая эпоха не смогла разлучить.
- Эта пластинка пережила пять рок-возрождений. Скоро будет шестое.
```

### [FACT_GENRE_CLASSICAL] — расширено
```
- Классика на виниле — отдельная религия. С аудиофилами не спорь.
- Эти прессинги делали аккуратнее всего — целевая аудитория была требовательная.
- Классика и джаз — два жанра, которые продали винил аудиофилам.
- Записи классики на Deutsche Grammophon — до сих пор эталон звукорежиссуры.
- Классические записи 60-70-х часто делались с минимумом микрофонов — две штуки в зале.
- На хорошем оборудовании классика на виниле даёт ощущение, что ты в зале.
- Динамический диапазон классики виниловый формат передаёт лучше, чем дешёвая цифра.
- Decca, EMI, DG — три кита классических виниловых издателей.
- Эта пластинка, скорее всего, прожила несколько хозяев. Классика передаётся по наследству.

### [FACT_GENRE_ELECTRONIC] — расширено
```
- Электроника на виниле — отдельный мир. Тут даже промахи звучат красиво.

- Electronic dance music и винил — близкие родственники, несмотря на цифровое происхождение жанра.
- Эта пластинка, возможно, крутилась в каком-то клубе ночью три года подряд.
- Электроника на виниле звучит мягче, чем на цифре — аналоговая тёплость съедает резкость синтов.
- Многие электронные музыканты до сих пор сводят альбомы под виниловый формат.

```

### [FACT_GENRE_HIPHOP] — расширено
```
- Хип-хоп вырос из винила. Без поворотников и баттлов диджеев его бы не было.
- На этих пластинках часто есть инструментальная сторона — для битмейкеров.
- Сэмплы в хип-хопе — это, по сути, кусочки чужих пластинок. Эта могла стать одним из них.
- Хип-хоп пластинки 80-90х выпускались с эксклюзивными инструменталками и а-капеллами для диджеев.
- Скретч на виниле — это разговор иглы с пластинкой. Цифра такого не умеет.
- Брейкбит, который качает любой хип-хоп — был придуман диджеями, крутившими пластинки в обе стороны.
- Crate digging — выражение из хип-хоп культуры. Поиск редких пластинок ради сэмплов.
- На обороте этой пластинки может быть жирный кусок, который кто-то уже засэмплировал.
- Хип-хоп вернул популярность тысячам забытых соул и фанк-пластинок 60-70-х.
- Хороший продюсер слышит сэмпл в трёх секундах винила, где другие слышат шум.
```

```
- Фанк. Если бы у винила был жанр-родитель, был бы фанк.
- Тяжёлый бас на этих пластинках — отдельный аттракцион.
- Фанк делался для танцпола. Громкость крути вверх.
- На фанковых пластинках басист — главный человек в группе.
- Грув в фанке — это математика. Один и тот же ритм, но точно по доле.
- James Brown переучил всех музыкантов своей группы играть «на единицу». До него никто так не делал.
- Фанк — основной источник хип-хоп сэмплов. Тысячи треков построены на этих ритмах.
- Эта пластинка где-то когда-то заставила людей танцевать. Скорее всего — много раз.
- Фанк живёт в басу и барабанах. Всё остальное — украшения.
- Никто не делал фанк так плотно и горячо, как делали его в 70-х на южных лейблах.
```

### [FACT_GENRE_SOUL] — расширено
```
- Соул. Голос важнее всего, и винил эту тёплость не съедает.
- Эту пластинку, скорее всего, ставили на бесчисленных свиданиях.
- Соул на виниле звучит так, как должен — мягко и в лицо.

### [FACT_GENRE_AMBIENT] — НОВЫЙ
```
- Эмбиент. Музыка, которая может играть, пока ты делаешь что-то другое.
- Брайан Ино придумал термин в 1978-м, но идея была старше.

- Эта музыка слышна периферическим слухом. Прямое внимание ей мешает.
- Эмбиент и виниловое шипение — почти один и тот же материал.
- Эту пластинку, возможно, никогда не слушали целиком за один раз.
- Эмбиент — единственный жанр, где «фоновая музыка» звучит как комплимент.
```

### [FACT_GENRE_KRAUTROCK] — НОВЫЙ
```
- Краут-рок. Немецкая сцена 70-х, которая повлияла на всю последующую электронику.
- Can, Neu!, Kraftwerk, Faust — четыре кита жанра.
- На этих пластинках часто один трек занимает всю сторону. И это нормально.
- Motorik beat — фирменный ритм краут-рока. Прямой, гипнотический, бесконечный.
- Без краут-рока не было бы пост-панка, индастриала и половины современной электроники.
- Эта пластинка, скорее всего, повлияла на больше музыкантов, чем продалась копий.
- Дэвид Боуи приезжал в Берлин в 70-х учиться у этой сцены. Его трилогия — оттуда.
- Краут-рок не любил названия. Сами музыканты предпочитали «коcмическая музыка».
- На виниле краут-рок звучит как должен — длинно, плотно, гипнотически.
- Эта пластинка — машина времени в немецкие студии 1972 года.
```

### [FACT_GENRE_INDIE_FOLK] — НОВЫЙ
```
- Инди-фолк. Жанр, который выжил между гитарой и тишиной.
- Эти пластинки часто записаны в чьём-то доме за зиму. И это слышно.
- Хороший инди-фолк требует винила больше, чем стрим. Шипение — часть атмосферы.
- Лоу-фай эстетика — намеренная. Идеальная запись в этом жанре звучит фальшиво.
- Эта пластинка, возможно, начиналась как личный дневник, а закончилась как альбом.
- Инди-фолк 2000-х вернул популярность акустической гитаре после двух десятилетий электроники.
- Эта музыка чаще играет ранним утром или поздним вечером. Не днём.
- Записи в этом жанре часто короче официальной длины — артист считает, что 30 минут уже достаточно.
- На этих пластинках голос редко идеален. Это сознательно.
- Хороший инди-фолк — это интимность, доступная только аналоговому формату.
```

### [FACT_GENRE_EXPERIMENTAL] — НОВЫЙ
```
- Экспериментальная музыка. Жанр, где правила писали те, кто их нарушал.
- Эту пластинку нельзя слушать невнимательно — она не для фона.
- Минимализм, конкретная музыка, авангард — всё это здесь рядом.
- John Cage, Steve Reich, Terry Riley — авторы, которые изменили представление о том, что такое музыка.
- Эта пластинка существует как идея в звуке. Обычные категории не подходят.
- На виниле такие записи звучат лучше — аналоговая нелинейность добавляет жизни даже к самой стерильной идее.
- Эту пластинку, возможно, продали в десятках экземпляров. Это нормально для жанра.
- Экспериментальные альбомы часто требуют чтения буклета — без контекста ничего не понять.
- Граница между шумом и музыкой здесь стирается. Это и есть смысл.
- Эта запись повлияла на музыкантов, которые потом продали миллионы. Сама — нет.
```

### [FACT_GENRE_WORLD] — НОВЫЙ
```
- Мировая музыка. Большая категория, которая объединяет всё, что Запад называет «не своим».
- Этот жанр на виниле часто аутентичнее любого стрима — пластинку записывали локально.
- Африканский фанк 70-х на виниле — отдельная сокровищница для коллекционеров.
- Эфиопский джаз, кубинский сон, бразильская тропикалия — всё это сейчас переоткрывают.
- Эту пластинку, возможно, привезли из страны, где её записали.
- Этно-музыкальные пластинки 60-70-х — антропологический документ, не только музыка.
- Лейбл Sublime Frequencies и Soundway переиздают эти забытые жемчужины. Спасибо им.
- Эта пластинка — встреча двух миров в одной бороздке.
- Многие мировые жанры обогатили рок и электронику. Услышишь — узнаешь.
- На виниле эта музыка ближе к своему оригиналу, чем на любом другом носителе.
```

# ЧАСТЬ 3. БАЗА ФАКТОВ

## РАЗДЕЛ А — 200 ОБЩИХ ФАКТОВ

> больше». Первые 100 — из v2. Следующие 100 — новые.

### Про винил как формат (1–25)
```
1. LP — long play. Формат, придуманный Columbia в 1948 году. До этого пластинки были на 78 оборотов и по 4 минуты на сторону.
2. Скорость 33⅓ — это компромисс. На 45 звук лучше, но меньше музыки на стороне. 33 выбрали ради длительности.
3. Стандартная пластинка имеет около километра бороздки. Игла проходит её за 20 минут стороны.
4. Винил — это, технически, поливинилхлорид. Тот же материал, из которого делают трубы.
5. На современных прессах вес 180 граммов — это аудиофильский стандарт. Звучит ровно так же, как 140, но стабильнее на проигрывателе.
6. Игла проигрывателя касается винила с давлением около 2 грамм. Это легче, чем кажется.
7. Внутренние треки на пластинке звучат хуже внешних. Это физика — на меньшем радиусе бороздка плотнее.
8. Поэтому хитовые треки часто ставят в начало стороны — там звук чище.
9. Stereo на виниле появился в конце 50-х. До этого всё было mono.
10. Mono-микс часто отличается от стерео-микса. Это разные мастеринги, а не «то же самое в одном канале».
11. Картонный конверт пластинки называется sleeve. Внутренний — inner sleeve.
12. Inner sleeve бывают с надписями (часто — тексты песен) или просто белые. Бумажные хуже — могут поцарапать.
13. Виниловые проигрыватели в 90-х почти исчезли. Сейчас их продают больше, чем 30 лет назад.
14. Цветной винил — это окрашенный пластик. На звук влияет минимально, на вид — сильно.
15. Прозрачный винил впервые сделали в 70-х. Это был маркетинговый ход.
16. Picture disc — пластинка с напечатанной картинкой на самом виниле. Звучит хуже обычной — из-за конструкции.
17. 7-дюймовый сингл крутится на 45 оборотах. Это формат поп-музыки 50-60х.
18. 10-дюймовый формат был популярен в 50-х, потом исчез. Сейчас иногда возвращается как arty-вариант.
19. 12-дюймовый сингл — изобретение электронной сцены 70-х. Один трек, длинный микс, для диджеев.
20. Половина массы пластинки — пустое пространство по краям и между треками. Только середина — звук.
21. На пластинке физически невозможно сделать больше 25 минут на стороне без потери качества.
22. Аналоговая запись на ленту в 1/2 дюйма — стандарт студий 60-70х. Сейчас редкость.
23. Lacquer — лаковый диск, с которого делают матрицу. Существует одни минуты. Из неё прессуют всю пластинку.
24. Direct-to-disc — запись без плёнки, прямо на лак. Самая чистая, но и самая дорогая технология.
25. Half-speed mastering — мастер делается на половинной скорости. Это даёт больше детализации в высоких.
```

### Про жанры и эпохи (26–50)
```
26. Альбом как высказывание появился в 60-х. До этого пластинки были сборниками синглов.
27. Sgt. Pepper's Lonely Hearts Club Band (1967) считается первым альбомом, задуманным как цельное произведение.
28. Концептуальный альбом — это альбом с общей темой через все треки. Pink Floyd довели идею до абсолюта.
29. Двойной альбом — пластинка из двух дисков. Появилась как способ дать музыкантам больше пространства.
30. Сторона A традиционно сильнее B — туда ставили хиты для радио.
31. Hidden track — спрятанный трек после длинной паузы. Изобретение CD-эры, но иногда встречается и на виниле.
32. Джаз и винил почти ровесники. Оба сформировались в 1940-50-х в нынешнем виде.
33. Бибоп в 40-х — первая эра, когда джаз начали записывать как «искусство», а не как танцевальную музыку.
34. Кул-джаз 50-х — реакция на бибоп. Спокойнее, медитативнее, белее по составу музыкантов.
35. Хард-боп — попытка вернуть джазу чёрные корни в конце 50-х. Blue Note это направление и поднял.
36. Фьюжн — слияние джаза с роком. Появился в конце 60-х, главные имена — Miles Davis и Weather Report.
37. Соул вырос из госпела, но без религиозных текстов. Главные сцены — Memphis (Stax) и Detroit (Motown).
38. Motown Sound — конвейерное производство хитов 60-х. Берри Горди буквально нанимал композиторов на ставку.
39. Фанк родился из соула в конце 60-х. James Brown — отец, Sly Stone — второй родитель.
40. Психоделический рок — реакция на 60-е. Обложки этой эпохи — отдельный музей графического дизайна.
41. Прог-рок в 70-х довёл идею «альбома как произведения» до 20-минутных треков с 8 частями.
42. Панк в 1977 был отрицанием прог-рока. «Три аккорда — и пошли».
43. Пост-панк — то, что выросло из панка, когда тот стал клише. Joy Division, Wire, PiL.
44. Нью-вейв — поп-версия пост-панка. Talking Heads, Blondie, Devo.
45. Синтвейв 80-х — момент, когда электроника стала массовой.
46. Грандж в начале 90-х убил американский глэм-метал за один год.
47. Хип-хоп начался в 1973 году с вечеринок Кул Хёрка в Бронксе.
48. Хип-хоп и винил — это любовь на всю жизнь. Без поворотников диджеев его бы не существовало.
49. Хаус родился в Чикаго в 80-х. Назван по клубу «Warehouse», где его впервые ставили.
50. Техно — это Детройт, конец 80-х. Хуан Аткинс, Деррик Мэй, Кевин Сондерсон.
```

### Про лейблы и звукозапись (51–75)
```
51. Blue Note был основан в 1939 году Альфредом Лайоном — немецким эмигрантом, влюблённым в джаз.
52. Звукоинженер Руди Ван Гелдер записал почти все классические альбомы Blue Note. Его студия в Нью-Джерси — святыня для джазфэнов.
53. Дизайнер Рид Майлз делал обложки Blue Note без музыкальных снимков — только графика и типографика. Это стало эстетикой эпохи.
54. ECM был основан Манфредом Айхером в 1969 году в Мюнхене. Он сам сводил каждый релиз.
55. Слоган ECM — «The most beautiful sound next to silence». Это не маркетинг, это техзадание.
56. Stax записывал в кинотеатре, переделанном в студию. Кривая акустика стала их фирменным звуком.
57. Sun Records открыл Элвиса, Джонни Кэша, Карла Перкинса и Джерри Ли Льюиса. В одном здании в Мемфисе.
58. Atlantic создали братья Эртеген в 1947 году — двое турков, влюблённых в чёрную музыку Америки.
59. Verve — лейбл Нормана Гранца, импресарио Эллы Фитцджеральд. Он первым в США платил джазменам как звёздам.
60. Impulse! назывался «The New Wave in Jazz». На нём вышли самые радикальные альбомы Coltrane.
61. Riverside — лейбл Билла Эванса. Делал тихий, интимный джаз для думающих.
62. Factory Records (Манчестер) дала миру Joy Division и New Order. Также — самый красивый провал в истории музыкальной индустрии.
63. 4AD задал визуальный язык 80-х — фотограф/дизайнер Vaughan Oliver сделал почти все их обложки.
64. Def Jam (Нью-Йорк) превратил хип-хоп из локальной субкультуры в глобальный жанр.
65. Sub Pop (Сиэтл) первым выпустил Nirvana. Bleach — это Sub Pop.
66. Rough Trade — индепендент-лейбл, открывший The Smiths и многих других. Существует до сих пор.
67. Studio One — главный лейбл регги. Без него не было бы Боба Марли в том виде, в котором мы его знаем.
68. Trojan Records издавал ямайскую музыку в Британии. Через них в Англию пришёл регги в 60-х.
69. Motown гордился тем, что их записи звучали даже из дешёвых радиоприёмников. Они проверяли треки на маленьких колонках.
70. Записи 50-х часто делались за один-два дубля. Студийное время стоило денег, и музыканты были готовы к работе.
71. Columbia Records — старейший работающий лейбл в мире. Основан в 1888 году в Вашингтоне.
72. Capitol Records здание в Голливуде построено в форме стопки пластинок. Намеренно.
73. Decca Records когда-то отказалась подписать The Beatles. Сказали: «Гитарные группы выходят из моды».
74. Apple Records — лейбл, основанный самими Beatles в 1968 году. Логотип — половинка зелёного яблока.
75. Warp Records (Шеффилд) — главный лейбл электроники 90-х. Aphex Twin, Autechre, Boards of Canada — все отсюда.
```

### Про коллекционирование (76–100)
```
76. Crate digging — рытьё в ящиках с пластинками в поисках сокровищ. Это и хобби, и поджанр культуры.
77. Самая дорогая пластинка в мире — единственная копия «Once Upon a Time in Shaolin» (Wu-Tang Clan). Продана за миллионы.
78. Дискогс (Discogs) — крупнейшая база данных винила в мире. Создана в 2000 году одним человеком, Кевином Льюандовски.
79. Виниловый ренессанс начался около 2007 года. С тех пор продажи растут каждый год.
80. Record Store Day — международный день виниловых магазинов, с 2008 года. Третья суббота апреля.
81. На Record Store Day выпускают эксклюзивные тиражи. За ними часто стоят очереди с 5 утра.
82. Грейдинг пластинок (M, NM, VG+, VG, G, F, P) — стандарт Goldmine. Используется всеми коллекционерами.
83. Mint (M) — пластинка, которую никогда не вынимали из конверта. Near Mint (NM) — играла, но идеально.
84. VG+ — лёгкие следы использования, но звук без артефактов. Это рабочее состояние для большинства коллекционеров.
85. G (Good) на самом деле значит «плохое». Парадокс грейдинга — «Good» это не «хорошо», а «играбельно».
86. Японские прессы 70-80-х часто звучат лучше американских оригиналов. Это правда, не миф.
87. Японцы прессовали в меньших тиражах и использовали более качественный винил.
88. Audiophile pressing — это пластинка, сделанная с особым вниманием к качеству. Часто 180g, half-speed mastered.
89. Bootleg — нелегальная пластинка, обычно с концертной записи. Многие культовые альбомы существуют только как bootleg.
90. Test pressing — пробный пресс перед массовым выпуском. Их делают 10-20 штук. Среди коллекционеров — желанная редкость.
91. White label — пластинка без печатного лейбла, обычно промо для радио. Тоже коллекционная штука.
92. Внутренний конверт может стоить дороже, чем сама пластинка. Особенно если он с автографами или редкими картинками.
93. Промо-копия часто отличается от тиражной — иногда меньше треков, иногда другой микс. Коллекционная ценность выше.
94. Audiophile-лейблы вроде Mobile Fidelity, Analogue Productions, Music Matters специализируются на «лучших возможных» переизданиях.
95. Original first pressing — главный объект охоты. Цена может быть в 10-100 раз выше переиздания.
96. Matrix number — выгравированный код на бороздке у этикетки. По нему опознают конкретный прессинг.
97. Dead wax — пространство между последней бороздкой и этикеткой. Там часто прячут сообщения — «Porky's prime cuts», «An afro», подписи.
98. Inner gatefold — раскрывающийся внутренний разворот двойного альбома. Целое произведение визуального искусства.
99. OBI strip — японская бумажная полоска, оборачивающая пластинку. Без неё цена сильно падает.
100. Halfspeed mastering, all-tube mastering, AAA (all-analog) — три флага качественного аудиофильского прессинга.
```

### Про звук и технику (101–125)
```
101. Стилус (игла) — расходник. В среднем меняется каждые 500-1000 часов воспроизведения.
102. Алмазная игла — стандарт. Сапфировая — устаревший вариант, изнашивается быстрее.
103. Сабвуферы и виниловые проигрыватели — плохие соседи. Низкие частоты вызывают резонанс иглы.
104. Прогрев колонок и техники для винила — не миф. После 30-50 часов звучание меняется к лучшему.
105. Хранить пластинки надо вертикально. Горизонтальное хранение приводит к деформации.
106. Прямой солнечный свет — враг винила номер один. Деформация и выцветание конверта обеспечены.
107. Идеальная температура для хранения — 18-22°C. Влажность — 40-50%.
108. Чистка пластинки перед каждым воспроизведением продлевает жизнь и иглы, и винила.
109. Антистатическая щётка — must have. Просто протереть рукавом — это царапины.
110. Замшевая щётка — для лёгкой пыли. Влажная чистка — для серьёзной.
111. RCM — record cleaning machine. Профессиональный аппарат для глубокой чистки. От €500 и выше.
112. Иногда «трески» на виниле — это не царапины, а статика. Лечится антистатиком.
113. Кат — это процесс резки матрицы. От качества ката зависит весь звук пластинки.
114. Мастеринг для винила и для цифры — разные процессы. Хорошие лейблы делают отдельный мастер для каждого формата.
115. RIAA-кривая — стандарт эквализации виниловых пластинок. Без неё винил звучал бы тонко и тихо.
116. Фонокорректор переворачивает RIAA-кривую обратно. Без него винил почти не слышно.
117. MM (moving magnet) и MC (moving coil) — два типа звукоснимателей. MC — лучше, но капризнее.
118. Идеальная скорость проигрывателя — точно 33⅓ или 45 оборотов. Отклонение слышно даже непрофи.
119. Стробоскоп — диск с точками для проверки точности скорости. Часто встречается на премиальных вертушках.
120. Tonearm должен быть параллелен пластинке. Угол атаки иглы — критичен.
121. VTA (vertical tracking angle) — настройка угла. Влияет на верха больше, чем на низы.
122. Anti-skating — компенсация бокового усилия на иглу. Без неё игла царапает.
123. Direct drive vs belt drive — два типа привода. Direct точнее, belt — мягче по характеру.
124. Plinth — основание проигрывателя. От его массы зависит, как он гасит вибрацию.
125. Изоляция вертушки от пола — отдельная наука. Кто-то ставит на бетонные блоки, кто-то — на песок.
```

   - Записан за две сессии. Почти всё с первого дубля.

   - Coltrane подал это как духовное приношение. Записал с классическим квартетом за одну сессию.
   - Альбом разделён на четыре части — это музыкальная молитва, не сборник треков.

   - Альбом нестандартных размеров. «Take Five» в 5/4 — невероятный хит для такой странной музыки.

4. Charles Mingus — Mingus Ah Um (1959)
   - Mingus писал быстро и злил всех вокруг. Этот альбом — концентрация его таланта.

5. Sonny Rollins — Saxophone Colossus (1956)
   - Тенор-саксофонист, который умел всё. Здесь — на пике.

   - Записан на Blue Note, хотя Кэннонбол был с Columbia. Майлз Дэвис тут как «сайдмен» — это лукавство, он в центре.

   - Monk играл, как никто другой. Кривые гармонии, точные паузы.

   - Концертная запись. Через десять дней после неё басист Скотт Лафаро погиб в аварии.

   - Школа Jazz Messengers выпустила половину звёзд джаза. Это её визитная карточка.

    - Хэнкоку было 25. Альбом про море, написанный за городом.

    - Shorter писал темы, как стихи — кратко и точно. Это одна из самых элегантных пластинок Blue Note.

    - Долфи играл на пределе языка джаза. Через несколько месяцев после записи он умер. Ему было 36.

    - Колман принёс фри-джаз. Половина мира признала, половина проклинала.

### Рок 60-х

```
31. The Beatles — Sgt. Pepper's Lonely Hearts Club Band (1967)
    - Альбом, который изменил представление о том, что такое альбом. Концептуальный, многослойный.
    - Обложка Питера Блейка с коллажом из 70 фотографий стоила $3000 — невиданно много для 1967.

32. The Beatles — Abbey Road (1969)
    - Последний альбом, записанный группой. Side B — почти непрерывная сюита.
    - Перекрёсток с обложки в Лондоне в 2010 был объявлен памятником культурного наследия.

33. The Beatles — Revolver (1966)
    - Многие считают именно его пиком Beatles, а не Pepper'а. «Tomorrow Never Knows» — первый психоделический хит.
    - Альбом записан за три месяца — короткий срок для такого новаторства. Группа репетировала прямо в студии.

34. The Beatles — The White Album (1968)
    - Двойной альбом, который чуть не разбил группу. Каждый писал свои треки отдельно.
    - Белая обложка с тиснёным номером экземпляра — каждая копия первого тиража имела свой номер.

35. The Beach Boys — Pet Sounds (1966)
    - Брайан Уилсон записал ответ Beatles. Beatles в ответ записали Pepper'а.
    - Уилсон не давал плёнкам выходить из студии — носил их с собой в карманах.

36. The Rolling Stones — Let It Bleed (1969)
    - Конец 60-х в звуке одного альбома. «Gimme Shelter» открывает его как пророчество.
    - Альбом вышел через несколько дней после трагедии в Альтамонте, где зрителя убил Hells Angel.

37. The Velvet Underground & Nico (1967)
    - Продано мало. Каждый купивший — основал группу. Так говорил Брайан Ино.
    - Энди Уорхол не только сделал обложку с бананом, но и числился продюсером — формально.

38. The Doors — The Doors (1967)
    - Дебют. «Light My Fire» и «The End» на одной пластинке.
    - «The End» Френсис Форд Коппола использовал в «Apocalypse Now» 12 лет спустя.

39. Pink Floyd — The Piper at the Gates of Dawn (1967)
    - Дебют с Сидом Барреттом. Через год Барретт сошёл с ума.
    - Название взято из главы книги «Ветер в ивах». Барретт любил английскую детскую литературу.

40. Jefferson Airplane — Surrealistic Pillow (1967)
    - Грейс Слик и психоделия Сан-Франциско. «White Rabbit» отсюда.
    - Дебютировал с этой группой Джерри Гарсия (потом — Grateful Dead) — он не пел, но числился «музыкальным консультантом».

41. The Jimi Hendrix Experience — Are You Experienced (1967)
    - Хендрикс приехал в Лондон неизвестным и через год был легендой. Этот альбом — точка перехода.
    - В американской и британской версиях альбома — разные треки. Британская — короче и сильнее.

42. Cream — Disraeli Gears (1967)
    - Клэптон, Брюс и Бэйкер. «Sunshine of Your Love» отсюда.
    - Название — оговорка одного из роуди, имевшего в виду «derailleur gears» — велосипедные переключатели.

43. The Who — Tommy (1969)
    - Рок-опера. Первая в своём роде. О слепом, глухом мальчике-чемпионе по пинболу.
    - Двойной альбом был адаптирован в фильм, в балет, в Бродвейский мюзикл. Самая монетизируемая рок-опера в истории.

44. The Band — Music from Big Pink (1968)
    - The Band приехали из Канады и записали в подвальном доме в Нью-Йорке.
    - Дом Big Pink в Вест-Сауджертисе действительно был розовым. Группа жила там вместе с Бобом Диланом.

45. Van Morrison — Astral Weeks (1968)
    - Морган записал за две сессии, по ночам, с джазменами. Альбом, который никто не понял в момент выхода.
    - Сейчас он входит во все списки великих альбомов. Тогда продал 20 тысяч копий.

46. Bob Dylan — Highway 61 Revisited (1965)
    - Дилан подключил гитару — и фолк-сцена его прокляла. «Like a Rolling Stone» — первый трек.
    - Когда «Like a Rolling Stone» вышел синглом, он длился 6 минут — невиданно для радио.

47. Bob Dylan — Blonde on Blonde (1966)
    - Первый двойной альбом в рок-музыке. Записан в Нэшвилле.
    - Дилан записывал с местными кантри-музыкантами — это был культурный шок для обеих сторон.

48. The Kinks — The Village Green Preservation Society (1968)
    - Тихий, английский, провальный в момент выхода. Сейчас — один из любимых альбомов знатоков.
    - Альбом — ностальгия по уходящей сельской Англии. Концептуально на десятилетия опередил время.

49. Love — Forever Changes (1967)
    - Артур Ли записал в Лос-Анджелесе в 1967 свой шедевр. Продано почти ничего. Сейчас — культ.
    - Альбом записан в апокалиптической атмосфере — Ли был уверен, что скоро умрёт. Прожил ещё 39 лет.

50. The Zombies — Odessey and Oracle (1968)
    - «Time of the Season» отсюда. Альбом записан за копейки и забыт.
    - Открыт заново в 90-х. Опечатка в названии («Odessey» вместо «Odyssey») осталась навсегда — её не стали исправлять.
```

### Рок 70-х

```
51. Led Zeppelin — Led Zeppelin IV (1971)
    - «Stairway to Heaven». Альбом без названия. Четыре символа на обложке.
    - Каждый символ выбрал участник группы для себя. Их значения официально никогда не раскрывались.

52. Led Zeppelin — Physical Graffiti (1975)
    - Двойной. «Kashmir» — главный трек.
    - На обложке — фотография реального дома в Нью-Йорке. Адрес 96–98 St. Mark's Place.

53. Pink Floyd — The Dark Side of the Moon (1973)
    - Альбом, который продаётся непрерывно с 1973 года. Концепт про время, безумие, деньги.
    - Был в чарте Billboard 200 на протяжении 956 недель — рекорд, не побитый до сих пор.

54. Pink Floyd — Wish You Were Here (1975)
    - Памяти Сида Барретта. Заглавная песня — одна из самых играемых на гитарах в мире.
    - Когда альбом записывался, в студию вошёл толстый лысый человек. Группа не сразу поняла, что это Сид Барретт.

55. Pink Floyd — The Wall (1979)
    - Двойной концептуальный альбом. О стене между человеком и миром.
    - Альбом превратили в концертное шоу со строящейся реальной стеной — самое масштабное шоу в рок-истории.

56. The Rolling Stones — Exile on Main St. (1972)
    - Двойной. Записан в подвале французской виллы, где Стоунзы прятались от британских налогов.
    - Условия записи были адскими — жара, наркотики, кривая акустика. Получилось гениально.

57. David Bowie — The Rise and Fall of Ziggy Stardust (1972)
    - Боуи придумал инопланетянина и стал им на полтора года. Альбом — биография этого персонажа.
    - Концепт-альбом про умирающего рок-героя. Боуи «убил» Зигги на сцене в 1973 году, чем шокировал группу.

58. David Bowie — Hunky Dory (1971)
    - Перед Зигги. Тёплый, мягкий, фортепианный.
    - «Life on Mars?» — Боуи писал, услышав «My Way» Синатры, и захотел сделать своё «My Way».

59. David Bowie — Low (1977)
    - Берлинская трилогия. Половина альбома — инструменталки. Записан с Брайаном Ино.
    - Боуи переехал в Берлин лечиться от кокаиновой зависимости. Сделал три великих альбома за два года.

60. Black Sabbath — Paranoid (1970)
    - Хеви-метал родился именно здесь. «Iron Man», «War Pigs», «Paranoid» — все на одной пластинке.
    - Заглавный трек был написан за 20 минут, чтобы добить альбом до нужной длины. Стал самым известным.

61. Deep Purple — Machine Head (1972)
    - «Smoke on the Water» отсюда. Записан в швейцарском казино, которое сгорело во время записи.
    - Песня — буквально документ. Группа описывает, как они смотрели на пожар казино с балкона отеля.

62. Queen — A Night at the Opera (1975)
    - «Bohemian Rhapsody». Самый дорогой альбом своего времени.
    - Альбом был назван в честь фильма братьев Маркс. Сами братья сначала были против, потом подружились с группой.

63. Stevie Wonder — Songs in the Key of Life (1976)
    - Двойной альбом с бонусным мини-альбомом. Стиви на абсолютной вершине.
    - Альбом записывали два года. Motown был в ярости — но получили один из самых продаваемых альбомов своего лейбла.

64. Stevie Wonder — Innervisions (1973)
    - «Higher Ground», «Living for the City». Альбом, на котором Стиви играл почти всё сам.
    - Через три дня после выхода альбома Стиви попал в тяжёлую автокатастрофу. Чудом выжил.

65. Marvin Gaye — What's Going On (1971)
    - Мотаун не хотел его выпускать. Гэй настоял. Один из главных соул-альбомов всех времён.
    - Гэй сам продюсировал — впервые для Motown. Берри Горди сначала отказывался, потом дал зелёный свет.

66. Joni Mitchell — Blue (1971)
    - Альбом, на котором Джонни обнажилась эмоционально. Слушать в плохой день — опасно.
    - Митчелл записала альбом в период личного кризиса. Она потом говорила, что чувствовала себя «прозрачной как стекло».

67. Joni Mitchell — Hejira (1976)
    - Здесь Джонни играет с джазменами, включая Жако Пасториуса. Альбом-путешествие.
    - Слово «Hejira» — арабское, означает «миграцию». Альбом писался во время автомобильных путешествий через США.

68. Carole King — Tapestry (1971)
    - 25 миллионов копий. Альбом, после которого песни-исповеди стали жанром.
    - До этого Кинг была автором песен для других. «Tapestry» — её первый альбом как солистки.

69. Neil Young — Harvest (1972)
    - «Heart of Gold». Самый продаваемый альбом Янга. И самый ненавистный для самого Янга.
    - Янг говорил, что «Heart of Gold» поставил его в середину дороги — и он немедленно свернул в кювет.

70. Neil Young — After the Gold Rush (1970)
    - До «Harvest». Тоньше, грустнее.
    - 17-летний Нил Лофгрин играл на пианино в нескольких треках. Потом стал гитаристом E Street Band.

71. The Eagles — Hotel California (1976)
    - Альбом-стандарт мягкого рока. Заглавная песня — про что — спорят до сих пор.
    - Двойное гитарное соло в конце трека Дон Фелдер и Джо Уолш играли по 16 нот в секунду. Записывали по 3 минуты за дубль.

72. Fleetwood Mac — Rumours (1977)
    - Группа разваливалась изнутри. Все спали со всеми не теми. Альбом получился великолепным.
    - 40 миллионов копий — один из самых продаваемых альбомов в истории. И весь — про чужие любовные истории внутри группы.

73. Bruce Springsteen — Born to Run (1975)
    - Альбом, на котором Спрингстин стал «Боссом». Эпичный рок Нью-Джерси.
    - Спрингстин работал над альбомом 14 месяцев и едва не сошёл с ума. Записал всё с нуля несколько раз.

74. Bruce Springsteen — Darkness on the Edge of Town (1978)
    - После «Born to Run» — тише, тяжелее, темнее.
    - Альбом задержали на три года из-за судебного процесса с менеджером. Спрингстин в это время выступал каждую ночь.

75. Steely Dan — Aja (1977)
    - Безупречно записанный альбом. Сессионные музыканты высшего класса.
    - Группа отбирала музыкантов как тренер бейсбольную команду. Иногда заставляли записывать соло по 30 раз.

76. Genesis — The Lamb Lies Down on Broadway (1974)
    - Последний альбом с Питером Гэбриэлом. Двойной концепт о Нью-Йорке.
    - Через тур после релиза Гэбриэл ушёл из группы. Никто не знал, что Genesis станет ещё успешнее с Коллинзом.

77. Yes — Close to the Edge (1972)
    - Прог-рок в чистом виде. Три трека, один — на всю сторону.
    - Запись была настолько изнурительной, что барабанщик Билл Брюфорд ушёл из группы сразу после.

78. King Crimson — In the Court of the Crimson King (1969)
    - С обложкой, которую не забыть. Прог-рок начался отсюда.
    - Создатель обложки Барри Годбер умер от сердечного приступа через год после выхода альбома. Ему было 24.

79. Frank Zappa — Hot Rats (1969)
    - Преимущественно инструментальный. Заппа здесь — гитарист и композитор, не сатирик.
    - На «Willie the Pimp» поёт Капитан Бифхарт — школьный друг Заппы.

80. Patti Smith — Horses (1975)
    - «Jesus died for somebody's sins but not mine». Первый трек, первая строка.
    - Обложка работы Роберта Мэпплторпа стала одной из самых узнаваемых в рок-фотографии.
```

### Панк и пост-панк

```
81. The Ramones — Ramones (1976)
    - Дебют. 14 треков за 29 минут. Все одинаковые. И прекрасные.
    - Альбом записан за неделю и стоил $6400. Дешевле, чем плита для звукозаписывающей студии того времени.

82. Sex Pistols — Never Mind the Bollocks (1977)
    - Единственный студийный альбом Pistols. Этого хватило, чтобы поменять музыку.
    - Слово «Bollocks» на обложке пытались запретить британские власти как непристойное. Суд оправдал группу.

83. The Clash — London Calling (1979)
    - Двойной альбом. Панк, который перерос панк — рок, рэгги, рокабилли.
    - Обложка пародирует первый альбом Элвиса Пресли — буквы те же, шрифт тот же, только разбивают гитару.

84. The Clash — The Clash (1977)
    - Дебют. Громче, проще, прямее.
    - Альбом записан за три выходных. Звук намеренно сырой — продюсер был против, группа настояла.

85. Wire — Pink Flag (1977)
    - 21 трек за 35 минут. Минималистичный панк. Альбом, который копировали 30 лет.
    - Многие треки длятся меньше минуты. Wire считали, что если идея сказана — нечего её повторять.

86. Television — Marquee Moon (1977)
    - Заглавный трек — 10 минут, и каждая нота на месте. Из Нью-Йорка, но не панк.
    - Том Верлен и Ричард Ллойд играли соло, переплетающиеся как разговор. Дуэт гитар, которому не было равных.

87. Joy Division — Unknown Pleasures (1979)
    - Обложка с пульсаром — самый растиражированный графический мотив в музыке.
    - Альбом записан за пять дней. Звукоинженер Мартин Хеннетт убрал из звука всё, что мог — оставил только суть.

88. Joy Division — Closer (1980)
    - Записан незадолго до самоубийства Иэна Кёртиса. Холодный, медленный, неизбежный.
    - Группа узнала о смерти Кёртиса, когда альбом уже был готов к печати. Изменить ничего не успели.

89. The Smiths — The Queen Is Dead (1986)
    - Моррисси и Марр в зените. Английский пост-панк-поп с цитатами на каждый случай.
    - Заглавный трек открывает альбом 7-минутной репликой королевы из фильма — это «There Is a Light That Never Goes Out».

90. The Smiths — The Smiths (1984)
    - Дебют. Гитары Джонни Марра уже здесь во всей сложности.
    - Группа записала альбом дважды — первая версия была отклонена лейблом как слишком сырая.

91. New Order — Power, Corruption & Lies (1983)
    - Joy Division без Кёртиса перешли на синтезаторы и танцпол.
    - Обложка работы Питера Сэвилла — репродукция картины Анри Фантен-Латура XIX века. Без указания группы и названия.

92. The Cure — Disintegration (1989)
    - Двойной альбом-погружение. Депрессивный, длинный, любимый.
    - Роберт Смит сделал альбом перед своим 30-летием, в депрессии. Лейбл считал, что это карьерное самоубийство.

93. Bauhaus — In the Flat Field (1980)
    - Готик-рок начался здесь.
    - Группа записала «Bela Lugosi's Dead» одним дублем за 6 минут. Песня длится 9 — её просто продолжали играть.

94. Siouxsie and the Banshees — Juju (1981)
    - Сьюзи на пике мрачной готики.
    - Группа начала как фанаты Sex Pistols, бывшие в первом ряду на их концертах. Сами научились играть позже.

95. Killing Joke — Killing Joke (1980)
    - Альбом, без которого не было бы Nirvana и Metallica. По их собственному признанию.
    - Курт Кобейн использовал басовую линию из их песни «Eighties» в «Come as You Are».

96. Public Image Ltd — Metal Box (1979)
    - Джонни Лайдон после Sex Pistols — холодный, дискомфортный, гениальный.
    - Первое издание шло в металлической коробке как у плёнки. Дорого для лейбла, культово для фанатов.

97. Talking Heads — Remain in Light (1980)
    - Записан с Брайаном Ино. Африканские ритмы встретились с нью-йоркским артом.
    - Группа записала фундаментальные грувы коллективной импровизацией, потом Бирн добавил тексты.

98. Talking Heads — Speaking in Tongues (1983)
    - «Burning Down the House» — большой хит. Альбом — переходный к танцполу.
    - На обложке первого издания — лицо Бирна на просвечивающем виниле. Очень дорого, очень редко.

99. Devo — Q: Are We Not Men? A: We Are Devo! (1978)
    - Странный, синтезаторный, в красных шлемах. Тоже Брайан Ино.
    - Группа исповедовала «теорию де-эволюции» — что человечество движется не вперёд, а назад.

100. Pixies — Doolittle (1989)
     - Кобейн признавался, что копировал именно их структуру тихо-громко.
     - Альбом продюсировал Гил Нортон. Стив Альбини, продюсер их дебюта, был категорически против — слишком чистый звук.
```

### Хип-хоп

```
101. Wu-Tang Clan — Enter the Wu-Tang (36 Chambers) (1993)
     - Звучит, как будто записано в подвале. Это и есть подвал. И это великолепно.
     - 9 человек подписали контракт с лейблом, но оставили за собой право выпускать сольные альбомы. Беспрецедентная сделка.

102. Nas — Illmatic (1994)
     - 39 минут. 10 треков. Каждый — шедевр. Дебют 20-летнего парня из Квинса.
     - На альбоме продюсировали DJ Premier, Pete Rock, Q-Tip, Large Professor — золотой век продюсирования в одном проекте.

103. The Notorious B.I.G. — Ready to Die (1994)
     - Дебют Бигги. Восточное побережье в высшей точке.
     - Через три года после выхода Бигги был застрелен в Лос-Анджелесе. Ему было 24.

104. 2Pac — All Eyez on Me (1996)
     - Двойной альбом. За год до его смерти.
     - Первый двойной хип-хоп альбом в истории. Записан за две недели после выхода Тупака из тюрьмы.

105. Dr. Dre — The Chronic (1992)
     - G-funk родился здесь. Снуп Догг дебютировал отсюда.
     - Альбом записан в подвале студии Death Row. Снуп тогда числился рэпером без контракта.

106. Snoop Dogg — Doggystyle (1993)
     - Дебют. Продолжение «The Chronic».
     - Альбом достиг #1 в Billboard сразу после выхода — первый дебютный альбом в истории, который сделал это.

107. A Tribe Called Quest — The Low End Theory (1991)
     - Хип-хоп, который пожал руку джазу. Слушается до сих пор.
     - На альбоме играет Рон Картер — легендарный джазовый контрабасист, игравший с Майлзом Дэвисом.

108. De La Soul — 3 Feet High and Rising (1989)
     - Хип-хоп до того, как стал серьёзным. Цветной, лёгкий, открытый.
     - Group использовала тысячи сэмплов без очистки. Их потом судили — победил истец, лейбл выплатил миллионы.

109. Public Enemy — It Takes a Nation of Millions to Hold Us Back (1988)
     - Самый громкий, самый политический хип-хоп-альбом эпохи.
     - The Bomb Squad — продюсерская команда альбома — наслаивала по 100-200 сэмплов в одном треке.

110. N.W.A — Straight Outta Compton (1988)
     - Гангста-рэп родился здесь.
     - ФБР отправило лейблу письмо с угрозой за песню «Fuck Tha Police». Группа использовала это для пиара.

111. Eric B. & Rakim — Paid in Full (1987)
     - Рэйким изобрёл многосложную рифмовку. До него такого не было.
     - До Рэйкима рэп был ритмическим — «там-та-там-там». Он принёс плотную поэзию.

112. Run-D.M.C. — Raising Hell (1986)
     - «Walk This Way» с Aerosmith. Хип-хоп вышел в мейнстрим.
     - Сотрудничество с Aerosmith организовал Rick Rubin. Aerosmith были в творческом кризисе — он спас их карьеру.

113. Beastie Boys — Paul's Boutique (1989)
     - Альбом из тысячи сэмплов. Сейчас такой нельзя было бы выпустить — лицензии съели бы бюджет.
     - Альбом продюсировали The Dust Brothers. Их подход — «звуковой коллаж» — повлиял на всё дальнейшее.

114. Beastie Boys — Licensed to Ill (1986)
     - Дебют. «Fight for Your Right» — первый рэп-хит белых ребят.
     - Первый рэп-альбом, достигший #1 в Billboard. Сейчас группа стыдится части текстов с этого альбома.

115. Outkast — Aquemini (1998)
     - Южный хип-хоп с шизофренической амбицией.
     - Названо в честь зодиаков членов группы — Andre 3000 (Водолей) и Big Boi (Близнецы).

116. Outkast — Stankonia (2000)
     - «B.O.B.» открывает альбом со скоростью 155 BPM. Хип-хоп так быстро никто не делал.
     - Стэнкония — название студии группы в Атланте. Они там работали без сна неделями.

117. Jay-Z — The Blueprint (2001)
     - Канье Уэст продюсировал треки. Выпущен 11 сентября 2001 года.
     - В день выхода террористы атаковали Нью-Йорк. Альбом всё равно вышел на #1.

118. Kanye West — The College Dropout (2004)
     - Канье как продюсер был известен. Как рэпер — нет. Этот альбом всё изменил.
     - Канье попал в страшную аварию незадолго до записи. Записал «Through the Wire» с проволокой в челюсти.

119. Kanye West — My Beautiful Dark Twisted Fantasy (2010)
     - Самый амбициозный хип-хоп-альбом 2010-х.
     - Канье снял дом на Гавайях и собрал десятки музыкантов и продюсеров. Запись напоминала художественную резиденцию.

120. Kendrick Lamar — Good Kid, M.A.A.D City (2012)
     - Кинематографичный альбом про взросление в Комптоне.
     - Альбом задуман как фильм — каждый трек как сцена, между ними — голосовые сообщения от мамы Кендрика.

121. Kendrick Lamar — To Pimp a Butterfly (2015)
     - Джаз, фанк, политика. Один из главных альбомов десятилетия.
     - Президент Обама пригласил Кендрика в Белый дом обсудить «To Pimp a Butterfly». Беспрецедентно для рэпера.

122. J Dilla — Donuts (2006)
     - Инструментальный хип-хоп. Записан Дилле в больнице, за три дня до его смерти.
     - 31 трек по 1-2 минуты — как короткие виньетки. Дилла знал, что умирает, и торопился.

123. Madvillain — Madvillainy (2004)
     - MF DOOM и Madlib. Альбом без хуков, без хитов, но с культовым статусом.
     - Утечка альбома произошла за год до релиза. Madlib и DOOM перезаписали половину после утечки.

124. Mos Def — Black on Both Sides (1999)
     - Сейчас он Yasiin Bey. Тогда — Mos Def с одним из самых поэтичных хип-хоп альбомов.
     - Mos Def — формальный поэт. Учился ремеслу у поэтов Last Poets и Гила Скотта-Херона.

125. The Roots — Things Fall Apart (1999)
     - Хип-хоп с живой группой. Долгий, медленный, аналоговый.
     - Это был первый альбом The Roots, который продался золотым тиражом. Группа существовала уже 12 лет.
```

### Соул, фанк, ритм-н-блюз

```
126. Aretha Franklin — I Never Loved a Man the Way I Love You (1967)
     - Запись в Muscle Shoals, легендарная сессия. Арета стала Аретой здесь.
     - Сессия завершилась дракой. Альбом дозаписывался в Нью-Йорке без южной ритм-секции.

127. Otis Redding — Otis Blue (1965)
     - Записан за 24 часа подряд. Хит за хитом.
     - Альбом включает кавер «Satisfaction» Rolling Stones — Реддинг записал её, не зная слов до конца, импровизируя.

128. Ray Charles — The Genius of Ray Charles (1959)
     - Здесь Чарльз соединил соул с биг-бэндом. Никто до него так не делал.
     - Альбом записан с двумя оркестрами — Quincy Jones аранжировал джазовую сторону.

129. Sam Cooke — Live at the Harlem Square Club, 1963
     - Кук был «королём соула». Эта концертная запись — самая горячая.
     - Альбом не выпускали 22 года. RCA считала, что «грубый» Кук на этой записи испортит его поп-имидж.

130. James Brown — Live at the Apollo (1963)
     - Концертный альбом. Записан с третьей попытки.
     - Брауну пришлось оплатить запись из своего кармана — лейбл считал концертный соул-альбом рискованным.

131. Curtis Mayfield — Super Fly (1972)
     - Саундтрек к фильму, который оказался лучше фильма. Социальный фанк.
     - Mayfield писал тексты как комментарий к сюжету фильма — критиковал героиновый бизнес, тогда как фильм его прославлял.

132. Stevie Wonder — Talking Book (1972)
     - «Superstition». Стиви начинает свою главную серию альбомов 70-х.
     - Заглавный сингл «Superstition» Стиви изначально написал для Джеффа Бека, но потом решил оставить себе.

133. Sly and the Family Stone — There's a Riot Goin' On (1971)
     - Тёмный, наркотический, гениальный. Через два года после оптимистичного «Stand!».
     - Sly Stone записывал альбом в одиночку, многократно записывая и стирая собственные дубли. Получилось мутно и гениально.

134. Funkadelic — Maggot Brain (1971)
     - Заглавный трек — 10 минут гитарного соло Эдди Хэйзела.
     - Хэйзел играл это соло один дубль. Джордж Клинтон сказал ему: «играй так, будто твоя мать только что умерла».

135. Parliament — Mothership Connection (1976)
     - П-Фанк во всей мощи. Космический фанк.
     - На концертах Parliament из «материнского корабля» спускался на сцену настоящий 3-тонный корабль. Шоу стоило целое состояние.

136. Earth, Wind & Fire — That's the Way of the World (1975)
     - Эпохальный соул-фанк-альбом. «Shining Star», «Reasons».
     - Альбом был саундтреком к фильму, который провалился. Альбом — нет.

137. Al Green — Call Me (1973)
     - Сладкий мемфисский соул. Эл Грин в безупречной форме.
     - Грин записывал у Уилли Митчелла в Royal Studios — деревянная комната, которой более 100 лет.

138. Donny Hathaway — Live (1972)
     - Один из лучших концертных альбомов в истории соула.
     - Хэтуэй страдал от шизофрении и покончил с собой в 33 года, выпрыгнув из окна нью-йоркского отеля.

139. D'Angelo — Voodoo (2000)
     - Нео-соул в его зените. Запись с The Roots и Q-Tip.
     - D'Angelo пропал после релиза на 14 лет — стал затворником, проиграл в весе и наркомании.

140. D'Angelo — Black Messiah (2014)
     - 14 лет после Voodoo. Слушается, как будто и не было паузы.
     - Альбом выпустили на месяц раньше плана из-за событий в Фергюсоне — он стал саундтреком к движению Black Lives Matter.

141. Erykah Badu — Mama's Gun (2000)
     - Эрика Баду на пике своего нео-соула.
     - Альбом начинается с 8-минутного трека «Penitentiary Philosophy» — никаких уступок поп-формату.

142. Lauryn Hill — The Miseducation of Lauryn Hill (1998)
     - Единственный её студийный альбом. Безупречный.
     - Альбом получил 5 «Грэмми» — рекорд для женщины-исполнителя на тот момент.

143. Frank Ocean — Channel Orange (2012)
     - Современный соул в самой утончённой форме.
     - За неделю до релиза Оушен написал в блоге об отношениях с мужчиной. Первый такой жест в хип-хоп/соул среде.

144. Frank Ocean — Blonde (2016)
     - Альбом, который Ocean делал в одиночку. Минималистичный, разрозненный, любимый.
     - Альбом вышел через 4 года после Channel Orange. Все эти годы Оушен молчал — даже о факте работы.

145. Solange — A Seat at the Table (2016)
     - Сестра Бейонсе сделала свой шедевр. Соул для современной эпохи.
     - Альбом был записан в Луизиане — Solange вернулась к корням семьи, чтобы записать его.
```

### Электроника

```
146. Kraftwerk — Trans-Europe Express (1977)
     - Немецкие пионеры синтезаторной музыки. Без них не было бы ни хип-хопа, ни техно.
     - «Trans-Europe Express» был сэмплирован в «Planet Rock» Африка Бамбата — основополагающем хип-хоп-треке.

147. Kraftwerk — Computer World (1981)
     - Альбом про эпоху, которая ещё не наступила. Сейчас он звучит как пророчество.
     - Группа писала про данные, экраны, банкоматы — за десятилетие до того, как это стало реальностью.

148. Brian Eno — Music for Airports (1978)
     - Эмбиент как жанр официально начался здесь.
     - Эно действительно хотел, чтобы эту музыку играли в аэропортах. В одном американском аэропорту её крутили несколько месяцев.

149. Brian Eno — Another Green World (1975)
     - Полу-песни, полу-инструменталки. Альбом-сад.
     - Эно сочинял на ходу, используя свои «косые карточки» — карточки с непонятными инструкциями для музыкантов.

150. Tangerine Dream — Phaedra (1974)
     - Берлинская школа электроники. Длинные, медленные путешествия.
     - Альбом записан на студии Virgin в Англии, в первые недели существования лейбла Брэнсона.

151. Aphex Twin — Selected Ambient Works 85-92 (1992)
     - Сборник домашних записей, ставший классикой.
     - Ричард Джеймс утверждал, что записал часть этих треков в 14 лет. Никто не смог это ни подтвердить, ни опровергнуть.

152. Aphex Twin — Richard D. James Album (1996)
     - Параноидальный, скоростной, человечный.
     - Обложка — собственное лицо Ричарда Джеймса. До этого его никто не видел публично много лет.

153. Boards of Canada — Music Has the Right to Children (1998)
     - Ностальгия по детству, которого не было. Эстетика 70-х видеозаписей.
     - Группа намеренно использовала старые синтезаторы и плёночные кассеты, чтобы получить «выцветший» звук.

154. Daft Punk — Discovery (2001)
     - Французский дом во всей пышности. «One More Time», «Harder, Better, Faster, Stronger».
     - Альбом был использован как саундтрек к аниме «Интерстелла 5555» — режиссёром выступил Leiji Matsumoto.

155. Daft Punk — Random Access Memories (2013)
     - Возвращение к живым инструментам. С Нилом Роджерсом, Джорджо Мородером.
     - Дуэт продал больше пластинок с этим альбомом, чем со всеми предыдущими вместе взятыми.

156. The Chemical Brothers — Dig Your Own Hole (1997)
     - Биг-бит на пике. Танцпол + рок-агрессия.
     - На «Setting Sun» поёт Ноэл Галлахер из Oasis. Сингл стал номером 1 в Британии.

157. Massive Attack — Mezzanine (1998)
     - Триждать-эстетика в чёрном цвете. Бристольский саунд.
     - Альбом записан настолько долго и тяжело, что состав группы развалился. Большинство альбома записали только два участника.

158. Massive Attack — Blue Lines (1991)
     - Первый трип-хоп-альбом. До этого жанра не существовало.
     - На альбоме поёт Шара Нельсон — её карьера началась здесь. Также — Тришка, тогда ещё не Tricky.

159. Portishead — Dummy (1994)
     - Бристоль, тяжёлый бит, голос Бэт Гиббонс. Триждать как жанр.
     - Группа выпускает альбомы крайне редко — за 30 лет только три. Их «Dummy» получил Mercury Prize.

160. Burial — Untrue (2007)
     - Дабстеп до того, как стал брокор. Тихий, городской, ночной.
     - Burial — псевдоним Уильяма Бевана. До 2008 года его личность была неизвестна — он был «лондонским тайным городским дабстеп-композитором».

161. Caribou — Swim (2010)
     - Танцевальная электроника, сделанная музыкантом-математиком.
     - Дэн Снейт (Caribou) имеет докторскую степень по математике от Имперского колледжа.

162. LCD Soundsystem — Sound of Silver (2007)
     - Танцпунк со средним возрастом и иронией.
     - Джеймс Мерфи начал группу в 33 — поздно для танцевальной музыки. Это и стало его темой.

163. Four Tet — Rounds (2003)
     - Folktronica. Электроника, звучащая как живая.
     - Кирен Хебден (Four Tet) сэмплирует свои собственные акустические записи — гитары, флейты, бубны.

164. Autechre — Tri Repetae (1995)
     - IDM в чистом виде. Сложно, абстрактно, требует внимания.
     - Альбом записан полностью на ранних драм-машинах и сэмплерах. Никаких компьютерных программ.

165. Underworld — Dubnobasswithmyheadman (1994)
     - Прогрессив-хаус с поэзией.
     - Эта пластинка переломила восприятие электронной музыки — впервые альбом дэнс-музыки слушали целиком, не для танцпола.
```

### Альтернативный рок и инди

```
166. Nirvana — Nevermind (1991)
     - Альбом, который убил глэм-метал и принёс гранж в мейнстрим.
     - Бутч Виг сделал звук альбома неожиданно чистым. Кобейн позже жалел — хотел сырого звука.

167. Nirvana — In Utero (1993)
     - После «Nevermind». Грубее, сырее. Кобейн хотел уйти от чистого звука Бутча Вига.
     - Альбом продюсировал Стив Альбини. Он сделал его за две недели, без эффектов, без полировки.

168. Pearl Jam — Ten (1991)
     - Дебют. Главный альбом гранжа, помимо «Nevermind».
     - Альбом стоял в Billboard 200 в течение 5 лет — невероятная стойкость для дебюта.

169. Soundgarden — Superunknown (1994)
     - «Black Hole Sun». Гранж с прог-роковой амбицией.
     - Альбом дошёл до #1 в Billboard в первую же неделю — необычно для тяжёлого жанра в 1994.

170. Alice in Chains — Dirt (1992)
     - Самый тёмный гранж-альбом. Стэйли поёт о наркотиках с ужасом изнутри.
     - Стэйли умер от передозировки в 2002. Многие тексты альбома оказались автопророчествами.

171. Radiohead — OK Computer (1997)
     - Альбом про будущее, написанный в 1997 году. Сейчас он звучит ещё актуальнее.
     - Альбом записан в особняке актрисы Джейн Сеймур в Англии. Группа жила там три месяца.

172. Radiohead — Kid A (2000)
     - Радиохед взорвали свой звук и пересобрали с нуля. Электроника, ноль гитарных хитов.
     - Альбом не имел синглов и почти не имел маркетинговой кампании. Всё равно стал #1.

173. Radiohead — In Rainbows (2007)
     - Группа выложила альбом «плати сколько хочешь». Революция в дистрибуции.
     - В среднем люди платили $6 за альбом. Радиохэду это принесло больше денег, чем все их предыдущие альбомы вместе.

174. Radiohead — The Bends (1995)
     - До OK Computer. Альбом, которым они доказали, что они не «Creep».
     - Йорк сорвал голос на гастролях после «Pablo Honey». «The Bends» он записал, восстанавливаясь.

175. R.E.M. — Automatic for the People (1992)
     - «Everybody Hurts», «Man on the Moon». Самый меланхоличный их альбом.
     - Название взято из лозунга кафе в родном городе группы Афинс, Джорджия. Кафе до сих пор работает.

176. R.E.M. — Murmur (1983)
     - Дебют. Бормотание Майкла Стайпа стало эстетикой.
     - Тексты Стайпа на ранних альбомах принципиально неразборчивы — он сам так хотел.

177. The Pixies — Surfer Rosa (1988)
     - До «Doolittle». Грубее, страннее. Записан Стивом Альбини.
     - Альбом был хитом в Британии, но почти никем не услышан в Америке. Это была судьба Pixies в 80-х.

178. My Bloody Valentine — Loveless (1991)
     - Стена гитарного шума, под которой — мелодии. Шугейз в высшей точке.
     - Запись стоила лейблу Creation $250,000 — невероятная сумма для инди в 1991. Чуть не разорила лейбл.

179. Sonic Youth — Daydream Nation (1988)
     - Двойной альбом. Эксперименты с гитарными настройками превратились в полноценные песни.
     - Каждая гитара группы настроена по-своему. Sonic Youth используют десятки «изобретённых» настроек.

180. Slint — Spiderland (1991)
     - Пост-рок начался здесь. Тихо, потом громко, потом снова тихо.
     - Группа распалась сразу после записи. Барабанщик попал в психиатрическую клинику, остальные ушли в обычную жизнь.

181. Pavement — Slanted and Enchanted (1992)
     - Лоу-фай как эстетика. Альбом, на котором инди стал инди.
     - Стивен Малкмус записывал альбом дома на 4-дорожечный магнитофон. Звук намеренно «дешёвый».

182. Beck — Odelay (1996)
     - Лоу-фай-хип-хоп-фолк-блюз. Бек смешал всё подряд и получилось.
     - Альбом продюсировали Dust Brothers — те же, что Beastie Boys «Paul's Boutique». Подход «звуковой коллаж».

183. The Strokes — Is This It (2001)
     - Альбом, который вернул рок-н-ролл в Нью-Йорк в нулевые.
     - Британская обложка альбома — фото руки на бедре женщины — была заменена в Америке на абстрактную графику.

184. The White Stripes — Elephant (2003)
     - Двое человек, гитара и барабаны. «Seven Nation Army».
     - Альбом записан полностью на аналоговое оборудование старше 1963 года. Принципиальная позиция Джека Уайта.

185. Arcade Fire — Funeral (2004)
     - Канадский инди в эпическом масштабе. Дебют.
     - Альбом назвали «Funeral», потому что во время записи в семьях членов группы умерло несколько родственников.

186. Modest Mouse — Good News for People Who Love Bad News (2004)
     - «Float On» вышел в радио. Альбом — гораздо страннее заглавного хита.
     - Группа существовала 10 лет до этого хита, в полной безвестности. «Float On» изменил всё за месяц.

187. Yeah Yeah Yeahs — Fever to Tell (2003)
     - Гаражный рок начала 2000-х в женском вокале.
     - Карен О пробежала через несколько отелей Нью-Йорка, разбивая свои гостиничные номера в туре. Стала иконой.

188. The National — Boxer (2007)
     - Меланхоличный американский инди для пьющих кофе утром.
     - Альбом включён в саундтрек предвыборной кампании Барака Обамы 2008. Группа симпатизировала.

189. Bon Iver — For Emma, Forever Ago (2007)
     - Записан в хижине в лесу за одну зиму. После расставания.
     - Джастин Вернон записал альбом полностью один, на простую звукозаписывающую установку. Без других людей.

190. Vampire Weekend — Vampire Weekend (2008)
     - Дебют. Африканский гитарный паттерн встретился с гарвардским образованием.
     - Группа познакомилась в Колумбийском университете, где все четверо были студентами.
```

### Современная классика

```
191. Amy Winehouse — Back to Black (2006)
     - Соул в современной упаковке. Винехаус не успела дожить до зрелости — это её зенит.
     - Альбом продюсировал Марк Ронсон с группой Dap-Kings — настоящей мемфисской ритм-секцией.

192. Adele — 21 (2011)
     - Самый продаваемый альбом 2010-х. Не случайно.
     - Альбом провёл в Top 5 Billboard 200 более 100 недель — больше, чем любой другой альбом 2010-х.

193. Arctic Monkeys — AM (2013)
     - Группа выросла из британской инди-сцены в стадион.
     - Алекс Тёрнер сменил образ — отрастил волосы, начал петь медленнее. Группа стала звучать «по-голливудски».

194. Tame Impala — Currents (2015)
     - Кевин Паркер один сделал главный психоделический поп-альбом десятилетия.
     - Паркер играет на каждом инструменте на альбоме. Tame Impala — это, по сути, один человек.

195. Mac DeMarco — Salad Days (2014)
     - Слекер-рок 2010-х. Расслабленный, ленивый, обаятельный.
     - ДеМарко записывает дома, на простой аппаратуре. Намеренно сохраняет лоу-фай эстетику.

196. King Krule — The OOZ (2017)
     - Голос подростка из юго-восточного Лондона, который звучит как взрослый ночной город.
     - Арчи Маршалл (King Krule) сменил несколько псевдонимов прежде, чем остановиться на этом.

197. FKA Twigs — LP1 (2014)
     - Альбом-сон. Электронный авангард с поп-доступностью.
     - До музыки FKA Twigs работала танцовщицей в клипах других артистов. Её движения — часть её музыкальной идентичности.

198. Beach House — Bloom (2012)
     - Дрим-поп в его самой узнаваемой форме.
     - Группа — дуэт из Балтимора. Они записывают почти всё дома, без сторонних продюсеров.

199. Sufjan Stevens — Carrie & Lowell (2015)
     - Альбом про маму, которая умерла. Самый тихий и личный из его работ.
     - Кэрри — мать Стивенса, ушедшая из семьи. Лоуэлл — отчим. Альбом — переработка детского опыта потери.

200. Big Thief — Two Hands (2019)
     - Современный американский фолк-рок. Записан вживую, без наложений.
     - Big Thief выпустили два альбома в 2019 году — «U.F.O.F.» и «Two Hands». Оба считаются великими.

### Регги, мировая, экспериментал, новые жанры

```
201. Bob Marley & The Wailers — Exodus (1977)
     - Альбом записан в Лондоне, куда Марли уехал после покушения на Ямайке.
     - Журнал Time назвал «Exodus» альбомом века в 1999 году. Спорное решение — но не безосновательное.

202. Bob Marley & The Wailers — Catch a Fire (1973)
     - Первый альбом Wailers на международном лейбле. Регги вышел из Ямайки.
     - Обложка первого тиража имела форму зажигалки Zippo, открывавшейся вертикально. Дорого и культово.

203. Burning Spear — Marcus Garvey (1975)
     - Регги о Маркусе Гарви — пророке движения растафари.
     - Burning Spear был одним из самых философских голосов регги. Альбом — манифест ямайского самосознания.

204. Lee «Scratch» Perry & The Upsetters — Super Ape (1976)
     - Даб-классика. Perry создал жанр и тут же его переопределил.
     - Студия Перри Black Ark сгорела дотла в 1979. Перри утверждал, что сжёг её сам — «изгонял дьявола».

205. Augustus Pablo — King Tubbys Meets Rockers Uptown (1976)
     - Pablo на мелодике, King Tubby за пультом. Даб в своей самой возвышенной форме.
     - Мелодика — игрушечный инструмент. Pablo сделал из неё голос целого жанра.

206. Black Sabbath — Master of Reality (1971)
     - Третий альбом за два года. Тяжелее предыдущих, плотнее.
     - Тони Айомми настроил гитару на полтора тона ниже — из-за травмы пальцев. Это создало звук дума и слюджа на десятилетия вперёд.

207. Iron Maiden — The Number of the Beast (1982)
     - Брюс Дикинсон присоединился к группе. Британский метал получил голос.
     - Альбом был объявлен сатанинским в США, но цифры в названии — отсылка к роману Хайнлайна, не к Откровению.

208. Metallica — Master of Puppets (1986)
     - Трэш-метал в зрелости. Восьмиминутные треки с тысячью гитарных партий.
     - Басист Клифф Бертон погиб в автокатастрофе вскоре после релиза. Группа решила продолжать.

209. Slayer — Reign in Blood (1986)
     - 29 минут чистой агрессии. Альбом, который определил жанр.
     - Альбом был отвергнут Columbia Records из-за текстов про Йозефа Менгеле. Вышел на Def Jam — лейбле хип-хопа.

210. Robert Johnson — King of the Delta Blues Singers (1961)
     - Сборник записей блюзмена, умершего в 1938 году. Все его треки — на одной пластинке.
     - Легенда гласит, что Джонсон продал душу дьяволу за умение играть на гитаре. Историки этому подвержены сомнению.

211. Muddy Waters — Folk Singer (1964)
     - Электрический блюз-гигант записал акустический альбом. Странное решение, гениальный результат.
     - Альбом задумывали как «для фолк-публики», которая тогда не воспринимала электрический блюз.

212. B.B. King — Live at the Regal (1965)
     - Концертный альбом, который объяснил остальному миру, что такое блюз.
     - Эта запись стала учебником для Эрика Клэптона, Майка Блумфилда, Стиви Рэя Вогана.

213. Johnny Cash — At Folsom Prison (1968)
     - Кэш выступил в тюрьме Фолсом. Запись стала классикой.
     - Кэш специально вставил в плейлист песни про убийства и заключение. Заключённые встретили его как своего.

214. Willie Nelson — Red Headed Stranger (1975)
     - Концептуальный кантри-альбом про беглого священника. Аскетичный, минималистичный.
     - Лейбл хотел отказаться выпускать — слишком странно. Альбом стал одним из самых продаваемых в кантри-истории.

215. Patsy Cline — Sentimentally Yours (1962)
     - Голос, который определил женский кантри. Заглавный трек — стандарт жанра.
     - Кляйн погибла в авиакатастрофе через год после релиза. Ей было 30.

216. Brian Eno — Ambient 1: Music for Airports (1978)
     - Эно изобрёл эмбиент как жанр. Этот альбом — манифест.
     - Эно дал каждой стороне название «1/1», «2/1» и т. д. — намеренно не называя треки именами.

217. Aphex Twin — Selected Ambient Works Volume II (1994)
     - Все треки без названий. Эмбиент в самой холодной форме.
     - Каждая обложка маленьких изображений — это идея конкретного трека. Ричард Джеймс ассоциировал звуки с картинками.

218. Stars of the Lid — The Tired Sounds of Stars of the Lid (2001)
     - Двойной альбом эмбиента. Долгий, медитативный, бесконечный.
     - Группа состоит из двух человек, живущих на разных континентах. Они обмениваются файлами по почте.

219. Can — Tago Mago (1971)
     - Краут-рок в полноте. Двойной альбом, на котором группа открыла «motorik» — гипнотический ритм.
     - Дамо Сузуки, японский певец Can, был «найден» уличным музыкантом в Мюнхене. Стал голосом группы.

220. Neu! — Neu! (1972)
     - Дебют дуэта из Дюссельдорфа. Изобрели motorik beat — прямой, постоянный ритм.
     - Без Neu! не было бы пост-панка, инди-рока, синтпопа. Они повлияли на всех.

221. Faust — Faust (1971)
     - Дебют немецкого авангардистов. Альбом записан в переоборудованной школе.
     - Лейбл Polydor выпустил альбом в прозрачной обёртке — внутри был прозрачный винил.

222. Kraftwerk — Autobahn (1974)
     - Первый коммерческий успех Kraftwerk. 22-минутный заглавный трек о немецкой автостраде.
     - Сокращённая версия трека стала хитом в США. Никто не знал, что слушает революцию.

223. Cluster — Zuckerzeit (1974)
     - Тёплый, мелодичный электронный альбом. Между авангардом и поп-музыкой.
     - Cluster позже записали два альбома совместно с Брайаном Ино. Те альбомы — ключевые для понимания эмбиента.

224. John Coltrane — Ascension (1965)
     - Фри-джазовый эпос. 40 минут коллективной импровизации с 11 музыкантами.
     - Альбом был выпущен в двух версиях — Edition I и Edition II — с разными мастерингами. Никто не знает почему.

225. Albert Ayler — Spiritual Unity (1965)
     - Фри-джаз в самой радикальной форме. Айлер играл так, как никто до и после.
     - Айлер был найден мёртвым в Ист-Ривере в 1970 году. Обстоятельства его смерти неясны до сих пор.

226. Fela Kuti — Zombie (1976)
     - Afrobeat достиг политической остроты. Альбом критикует нигерийскую военную диктатуру.
     - Через год после релиза военные сожгли дом Кути и убили его мать. Сам он провёл много лет в тюрьме.

227. Tinariwen — Aman Iman: Water Is Life (2007)
     - Туареги Сахары играют гитарный рок. Уникальный голос пустыни.
     - Группа состоит из бывших солдат повстанческой армии. Их музыка — голос целого народа.

228. Mulatu Astatke — Ethio-Jazz (1974)
     - Эфиопский джаз — отдельный жанр, придуманный одним человеком.
     - Астатке учился музыке в Лондоне и Бостоне, потом вернулся в Эфиопию и создал жанр почти в одиночку.

229. Caetano Veloso — Caetano Veloso (1968)
     - Бразильская tropicalia в дебютном альбоме. Революция в латинской музыке.
     - Через год после релиза Велозу был арестован военной хунтой и выслан из Бразилии.

230. Os Mutantes — Os Mutantes (1968)
     - Бразильский психоделический рок. Странный, цветной, новаторский.
     - Группа стала культовой через 30 лет после распада. Курт Кобейн пытался их воссоединить — не успел.

231. John Cage — Imaginary Landscape No. 4 (запись 1962)
     - Композиция для 12 радиоприёмников. Случайность как метод сочинения.
     - Кейдж установил частоты и громкость, но содержание зависело от того, что транслировали в момент исполнения.

232. Steve Reich — Music for 18 Musicians (1978)
     - Минимализм в его самой богатой форме. 55 минут пульсирующего звука.
     - Альбом записан с одного дубля. Музыканты репетировали год, прежде чем приступить к записи.

233. Terry Riley — A Rainbow in Curved Air (1969)
     - Электронный минимализм до того, как стал жанром. Альбом из двух длинных треков.
     - Альбом повлиял на Кейрел, Pink Floyd, краут-рок. Райли работал в одиночку, импровизируя в реальном времени.

234. Philip Glass — Glassworks (1982)
     - Минимализм для широкой аудитории. Восемь коротких пьес.
     - Гласс продюсировал альбом сам и записал на свой лейбл, потому что классические лейблы отказывались.

235. La Monte Young — The Well-Tuned Piano (1981)
     - Дронам музыка на 5-часовом фортепианном произведении. Самое длинное классическое сочинение в истории.
     - Янг работал над пьесой 30 лет до этой записи. Каждое исполнение длилось пять часов.

236. Throbbing Gristle — 20 Jazz Funk Greats (1979)
     - Альбом, который выглядит как лёгкая поп-музыка, а звучит как индустриальный кошмар.
     - Обложка с фотографией группы на солнечном склоне холма — сознательная ирония. Холм этот — известное место самоубийств в Англии.

237. Coil — Horse Rotorvator (1986)
     - Индустриал с уклоном в оккультизм. Один из самых сложных альбомов жанра.
     - Группа состояла из двух человек, бывших партнёров в жизни. Их работа неразделима от их отношений.

238. Whitehouse — Bird Seed (2003)
     - Power electronics в чистом виде. Шум как форма музыки.
     - Альбом записан как «терапия» — участники группы пытались физически выразить психологическую агрессию.

239. Merzbow — Pulse Demon (1996)
     - Японский ноиз во всей своей физической силе.
     - Merzbow — один человек, Масами Акита, выпустил более 400 альбомов. «Pulse Demon» считается самым доступным.

240. Sufjan Stevens — Illinois (2005)
     - Концептуальный альбом про штат Иллинойс. Стивенс планировал сделать альбом про каждый штат — отказался после второго.
     - На альбоме 22 трека, более 70 минут. Стивенс играет на десятках инструментов.

241. Sigur Rós — Ágætis byrjun (1999)
     - Исландский пост-рок. Альбом, который привлёк мировое внимание к малой северной стране.
     - Йонси Биркьюр (вокалист) поёт на смеси исландского и придуманного языка «Hopelandic».

242. Godspeed You! Black Emperor — Lift Your Skinny Fists Like Antennas to Heaven (2000)
     - Двойной альбом пост-рока. Четыре эпических трека, каждый длится более 20 минут.
     - Группа никогда не даёт интервью. Их музыка — единственный язык, на котором они общаются с миром.

243. Mogwai — Young Team (1997)
     - Шотландский пост-рок. Альбом, на котором жанр получил форму.
     - Группа известна тем, что играет музыку без слов — но называет треки фразами, которые мог бы петь вокалист.

244. Tortoise — TNT (1998)
     - Чикагский пост-рок. Джаз, электроника, рок — слились в один жанр.
     - Tortoise состоит из музыкантов, которые играют в десятке других проектов. «TNT» — их вершина.

245. Slowdive — Souvlaki (1993)
     - Шугейз во второй волне. Альбом, который провалился в 1993, переоткрыт в 2010-х.
     - Slowdive распались сразу после альбома, испытывая разочарование. Воссоединились через 20 лет.

246. Cocteau Twins — Heaven or Las Vegas (1990)
     - Дрим-поп в самой узнаваемой форме. Голос Лиз Фрейзер — отдельный инструмент.
     - Лиз Фрейзер поёт фактически на придуманном языке. Слова неразборчивы намеренно.

247. The Jesus and Mary Chain — Psychocandy (1985)
     - Поп-песни, утопленные в гитарном фидбэке. Альбом, изобретший шугейз.
     - Концерты группы длились по 20 минут. Они играли так громко и так агрессивно, что часто слушатели уходили.

248. My Bloody Valentine — Isn't Anything (1988)
     - До Loveless. Дебютный альбом, на котором группа уже придумала свой звук.
     - Альбом записан за 3 недели. Loveless — за 2 года и 19 студий.

249. The Stone Roses — The Stone Roses (1989)
     - Манчестерский «мэдчестер» в дебютном альбоме. Соединил рок с танцполом.
     - Группа судилась с лейблом столько, что не выпускала альбома 5 лет. Когда выпустили «Second Coming» — уже было поздно.

250. Spiritualized — Ladies and Gentlemen We Are Floating in Space (1997)
     - Альбом-таблетка. Первый тираж был выпущен в форме блистера для лекарств.
     - Джейсон Пирс написал альбом после расставания с многолетней партнёршей. Альбом — обряд прощания.
```

### Поп и эстрадная классика

```
251. Michael Jackson — Thriller (1982)
     - Самый продаваемый альбом в истории. Около 70 миллионов копий.
     - Винсент Прайс записал монолог в финале заглавного трека за один дубль. Quincy Jones заплатил ему $20,000.

252. Michael Jackson — Off the Wall (1979)
     - До «Thriller». Quincy Jones как продюсер на пике формы.
     - Джексону был 21 год. Альбом ознаменовал его переход из подросткового артиста во взрослого.

253. Prince — Purple Rain (1984)
     - Соулдрак к фильму, который оказался саундтреком к десятилетию.

# ЧАСТЬ 4. ИНСТРУКЦИИ ПО СВЯЗКЕ С ПРИЛОЖЕНИЕМ

### Применение новой логики

Когда пользователь добавляет пластинку:

1. Запросить Discogs и получить: artist, title, year, label, genre.
2. Применить **новую логику рандомизации** из инструкций в начале файла:
   собрать взвешенный пул, вычесть недавно показанные, выбрать случайно.
3. Если в Разделе Б есть совпадение по artist+title — у этой пластинки 2 факта.
   При первом совпадении тянуть случайный из двух (или первый, если важна
   последовательность). При повторе — другой.
4. Если совпадения нет — фоллбэк на общие массивы по приоритету и весу.
5. Записать ID показанного факта в «недавно показанные» (max 20).

### Запасные правила

- Если для нового жанра (регги, металл, блюз и т.д.) Discogs возвращает
  жанр в английском написании — нормализовать перед сопоставлением с
  массивами.
- Если у пластинки несколько релевантных жанров (например, «джаз/фьюжн») —
  использовать все, не только первый.
- Эпоху определять по году релиза, не по году тура.
